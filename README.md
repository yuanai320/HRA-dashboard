# HRA 群体健康洞察仪表盘 v1.0

> 基于 **396 份真实 HRA 健康风险评估数据** 的群体健康洞察工具，对应「数字健康管理 / 健康大数据」专业方向。
> 作者：袁爱（东北大学秦皇岛分校 健康服务与管理专业 120410T）

## 这个作品解决什么问题
单个 HRA 报告只能看「一个人」，但养老机构 / 体检中心 / 健康企业更需要看「一群人」——
哪类人群最弱、哪个系统最该优先干预、资源该往哪投。本仪表盘把 396 份真实检测数据
（85 个身体区域功能偏离值）聚合为 6 张可交互图表 + 专业解读，一眼看清群体健康趋势。

## 6 张图
1. **群体健康指数分布** —— 整体健康水平与档位占比
2. **各系统异常率排行** —— 干预优先级（消化 / 呼吸 / 骨骼居前）
3. **年龄分层对比** —— 识别「中年塌陷」风险人群
4. **性别差异** —— 男女健康弱点不同
5. **身体区域偏离 TOP 10** —— 最弱器官精准定位
6. **系统异常联动热力图** —— 健康问题「打包」规律

## 指标说明
- 身体区域功能偏离值：HRA 对 85 个区域逐项检测，绝对值越大越异常。
- 异常参考线：`|偏离| ≥ 30` 记为「需关注」（HRA 通用判读惯例，改 `app.py` 中 `THRESH` 可调）。
- 综合健康指数（派生）：`100 − 平均区域绝对偏离 ×(100/60)`，用于群体横向对比，**非临床诊断**。

## 数据隐私
`data.xlsx` 已由 `gen_data.py` 脱敏（删除姓名 / 会员号列），仅用于群体统计，不含个人身份。

## 本地运行
```bash
pip install -r requirements.txt
python gen_data.py        # 生成本地脱敏 data.xlsx（首次）
streamlit run app.py
```

## 部署到 Streamlit Cloud（免费永久在线）
1. 把本目录（app.py / data.xls / requirements.txt / .streamlit/）推到 GitHub Public 仓库。
2. 打开 https://streamlit.io/cloud → New app → 选该仓库的 app.py。
3. 等待部署，获得永久在线链接。
